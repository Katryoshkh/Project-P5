Sistem Informasi Sekolah

Ini adalah tampilan project kelompok 1.

Sistem Informasi Sekolah ini dirancang untuk membantu pengelolaan data siswa, guru, dan jadwal pelajaran secara efisien dan terpusat. Melalui aplikasi ini, pihak sekolah dapat memasukkan, mengedit, dan melihat informasi penting terkait aktivitas akademik harian. Selain itu, terdapat fitur pengumuman dan dashboard statistik sederhana untuk mendukung manajemen sekolah berbasis digital.

Dalam pembuatan project ini, kami menggunakan HTML dan CSS untuk antarmuka pengguna, serta struktur halaman yang responsif. Project ini bertujuan sebagai representasi digitalisasi manajemen sekolah yang dapat dikembangkan lebih lanjut menggunakan backend seperti PHP atau database seperti MySQL di masa depan.
