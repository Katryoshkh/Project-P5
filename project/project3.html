Smoke & Flame Detector

Project Kelompok 3 - Kelas 11 Saintek 4

Deskripsi Singkat:
Smoke & Flame Detector adalah sistem pendeteksi asap dan api berbasis Arduino UNO yang dirancang untuk mengidentifikasi aktivitas seperti merokok atau membakar korek api di area rawan seperti kamar mandi sekolah. Sistem ini menggunakan sensor MQ-2 dan flame sensor untuk mendeteksi keberadaan asap atau api, kemudian memberikan peringatan melalui buzzer dan menampilkan status pada layar LCD.

Tujuan Proyek:
- Mendeteksi asap rokok atau api korek di area rawan.
- Mencegah tindakan menyimpang seperti merokok oleh siswa.
- Memberikan peringatan dini secara otomatis.

Komponen yang Digunakan:
- Arduino UNO
- Sensor MQ-2 (asap/gas)
- Flame Sensor (api)
- LCD 16×2
- Buzzer Passive
- Breadboard & Kabel Jumper
- Casing dari wadah tempat makan bekas (dicat untuk tampilan menarik)

Cara Kerja Sistem:
Sensor MQ-2 dan flame sensor akan memantau udara dan lingkungan sekitar. Jika salah satu atau keduanya mendeteksi asap atau api, Arduino akan mengaktifkan buzzer dengan suara panjang sebagai alarm dan menampilkan status deteksi di layar LCD. Sistem akan terus aktif hingga kondisi kembali normal.

Manfaat Sistem:
- Mendeteksi dini potensi pelanggaran atau bahaya kebakaran.
- Meningkatkan pengawasan di area tersembunyi.
- Mudah dirakit, hemat biaya, dan ramah lingkungan berkat penggunaan barang bekas sebagai casing.
